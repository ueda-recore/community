# RECORE Community

RECOREコミュニティの公式ランディングページです。リユース業界の経営者が集まり、知識・経験・課題を共有するコミュニティの思想と、関東・関西地域別のコミュニティについて紹介しています。

## 概要

- **メインサイト**: RECOREコミュニティの全体的な思想、ビジョン、ロードマップを紹介
- **関東コミュニティページ**: 関東地域のコミュニティについての情報
- **関西コミュニティページ**: 関西地域のコミュニティについての情報

## セクション構成

### メインサイト (index.html)

1. **ヘッダー**: ロゴとナビゲーション
2. **ヒーロー**: メインメッセージ「孤独な試行錯誤を終わらせる」と地域別リンク
3. **コミュニティについて**: コミュニティの定義と5つのメリット
4. **ビジョン**: 「共有」「解決」「共創」「信頼」の4つの価値観
5. **ロードマップ**: Phase 1～3の進化計画
6. **なぜ、RECOREはコミュニティを作るのか**: RECORE側の思想
7. **参加方法**: 参加資格、費用、開催頻度、お問い合わせ

### 地域別ページ

各地域ページは以下を含みます：
- 地域固有のコミュニティ紹介
- 直近のイベント情報
- 地域の思想説明
- 参加申し込み

## GitHub Pages デプロイ

1. このリポジトリの Settings > Pages にアクセス
2. Source を「Deploy from a branch」に設定
3. Branch を「main」、フォルダを「/ (root)」に設定
4. 保存して数分待つとサイトが公開されます

## 開発・カスタマイズ

- HTMLファイルはセマンティックな構造を使用しています
- CSSはCSS変数で色を管理しており、簡単にテーマ変更可能です
- JavaScriptはスムーススクロールとセクションのフェードイン機能を提供します
